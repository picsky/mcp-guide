# 概述篇：认识 MCP

## 什么是 MCP？

**Model Context Protocol (MCP)** 是由 **Anthropic** 于 **2024年11月** 开源的 AI 与外部世界连接的统一标准协议。

> **简单理解**：MCP 是 "AI 的 USB-C 接口"

## 核心价值

- 🚀 **标准化**：统一协议，一次开发多处使用
- 🔒 **安全性**：内置权限控制和验证机制
- 🔄 **可维护性**：独立更新不互相影响
- 🌐 **生态共享**：全球开发者共建工具库

## 核心组件

| 组件 | 作用 |
|------|------|
| **Tools** | AI 可调用的函数 |
| **Resources** | AI 可读取的数据 |
| **Prompts** | 预定义的交互模板 |

## 应用场景

- AI 助手增强
- 开发工具集成
- 数据分析
- 自动化工作流
- 知识管理
